4 .  수맞추기 게임  ver2   
 컴퓨터가 랜덤수를 정합니다. 
 컴퓨터가 랜덤수를 맞추는 게임입니다.
 세 개의 변수가  랜덤수를 맞춥니다. 맞출 때 까지 반복합니다
 세 개의 변수 중에  어떤 변수가  맞췄는지를 사용자가 선택하는 게임입니다.

1단계

정답을 변수 a라고 정한다.
변수 a의 범위는 문제 3번과 같이 1~100이라고 정한다.

컴퓨터가 뽑을 3개의 랜덤수는 각각 b, c, d라고 정한다.
b와 c와 d는 같은 같은 숫자가 저장될 수 없음을 정의한다.


2단계

모든 변수를 0으로 초기화한다.
컴퓨터는 1~100의 숫자중 하나의 숫자를 난수알고리즘을 통해 선택하여 변수 a에 저장한다.

이후 컴퓨터는 변수 a를 b, c, d중 랜덤한 변수 하나에에 저장한다.
남은 두 변수는 난수알고리즘으로 숫자를 선택하여 각각 저장한다.
만약 남은 두변수가 a와 같은 같은 숫자를 뽑은 경우 숫자를 다시 선택하여 저장한다.

예를 들어 변수 a가 50이고 변수 c=a가 되어 c=50이 되었다. 그리고 b, d는 각각 28, 71이 저장되었다.

3단계

플레이어는 변수 b, c, d만 선택이 가능하다.
플레이에게는 각 변수에 저장된 상수만 노출한다.

플레이어 선택 변수에 저장된 상수가 a에 저장된 상수와 같은지 확인한다.
만약 동일하다면 정답을 맞췄습니다.
틀렸다면 틀렸습니다. 문구를 출력한다.


4단계
만약 정답을 맞췄다면 2단계를 재 실행한다.
틀렸다면 선택지에서 틀린 변수를 제외하고 나머지 변수만 선택가능하도록 한다.

정답을 맞췄다면 정답을 맞췄습니다. 문구 출력후 2단계 재실행.
틀렸다면 틀렸습니다 문구 출력.
남은 변수는 반드시 정답이므로 맞추지 못했습니다.
정답은 (a에 저장된 상수를 노출)입니다. 츨력 및 
다음세션으로 문구 출력.

이후 2단계 재실행




5. 자신의 나이구하기

1단계
플레이어 나이를 저장할 변수를 a라고 한다.

이후 태어난 년도를 변수 b
태어난 달을 변수 c
태어난 날짜를 변수 d로 정의한다.

현재 년도를 변수 e
현재 달을 변수 f
현재 날을 변수 g 로 정의한다.


2단계

사용자는 총 4자리 상수를 입력해야하는 변수 b에 숫자를 기입한다.
사용자는 총 2자리까지 입력 가능한 변수 c에 상수를 기입한다.
사용자는 총 2자리까지 입력 가능한 변수 d에 상수를 기입한다.

컴퓨터는 현재날짜를 구해 위와 동일하게 변수 e, f, g에 각각 저장한다.

3단계
e-b를 수행하여 나온 상수를 a에 저장한다.

이후 사용자가 만나이를 넘겼는지 계산해야한다.
만약 c>f 라면 a에 -1를 하여 a에 저장한다. 이는 생일이 지나지 않은 사용자의 나이다.
당신의 나이는 만 a입니다 출력.

4단계
만약 c<f라면 태어난 달을 초과했으므로 a에 저장된 상수를 그대로 출력한다.
이는 만나이인 플레이어의 나이다.
당신의 나이는 a입니다 출력.

5단계
만약 c=f라면 d>=g인지 확인한다. 만약
d>=g일경우 생일이 지나지 않았으므로 a-1하여 a에 저장한다.
당신의 나이는 만a입니다. 출력

6단계
d<g일 경우 만나이이므로 a를 그대로 사용한다.
당신의 나이는 만 a입니다 출력.
